# Melampo Package

`src/melampo/` is the new baseline package for Project Melampo.

It replaces the old prototype-oriented layout with a more maintainable structure organized around:

- clinical interoperability
- data ingestion and normalization
- models and optional theoretical-quantum research layers
- memory and retrieval
- reasoning and metacognitive control
- training and generative replay
- evaluation and falsification
- orchestration and external service routing

All new implementation work should target this package.
